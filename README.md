# Manager

Manager is the local workflow repo for Assistant Elon Ma. Start with [START_HERE.md](START_HERE.md) to configure your own project path, chat bridge credentials, LLM provider profile, `.env.local`, and role profile mapping.

The setup guide is intentionally provider-agnostic: keep real secrets in `.env.local` or system environment variables, and put only local project paths and env variable names in local config files.

New to command-line setup? Use [START_HERE_FOR_BEGINNERS.md](START_HERE_FOR_BEGINNERS.md).

Giving this repo to an AI coding agent? Use [docs/agent-setup-guide.md](docs/agent-setup-guide.md).
